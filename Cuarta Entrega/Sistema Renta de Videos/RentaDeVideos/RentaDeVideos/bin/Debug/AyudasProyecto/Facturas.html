Ayuda: Facturas


Acceso al formulario Facturas

El formulario facturas esta disponible en la barra de menu, al dale click escoger el aparatado de "Ingresar". A continuación se le abrirá una ventana centrada, dentro de este formulario debe llenar los campos requeridos para su posterior ingreso a los datos del sistema.




Vista del Formulario de Ingreso de Facturas

Como en se muestra en la siguiente imagen, el formulario cuenta con los campos de ingreso No Factura, Fecha y ID Cliente, Tipo de Documento y tipo de pago. Y un grid donde se ingresa las lineas de productos rentados por el cliente.




Ingreso de datos

Se le solicita el ingreso de No. de Factura, No. Serie, ID Cliente, Tipo de documento, Forma de Pago y Fecha. Los datos de cliente se llenan automaticamente al seleccionar el cliente. Posteriormente dentro del grid debe llenar solamente el video a rentar y la cantidad, el sistema hace los respectivos cálculos para la transacción.




De click en el botón "Registrar Factura". Si el ingreso de datos es exitoso, se le mostrará un mensaje como el de la siguiente imagen.
