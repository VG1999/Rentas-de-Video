Ayuda: Compras


Acceso al formulario Compras

El formulario compras esta disponible en la barra de menu, al dale click escoger el aparatado de "Ingresar". A continuación se le abrirá una ventana centrada, dentro de este formulario debe llenar los campos requeridos para su posterior ingreso a los datos del sistema.




Vista del Formulario de Ingreso de Compras

Como en se muestra en la siguiente imagen, el formulario cuenta con los campos de ingreso No Compra, Fecha y ID Proveedor, el campo NIT automático. Y un grid donde se ingresa las lineas de productos comprados al proveedor.




Ingreso de datos

Se le solicita el ingreso de No Compra, Fecha y ID Proveedor. El campo de NIT se llenará automáticamente cuando seleccione el proveedor. Posteriormente dentro del grid debe llenar solamente el video a comprar y la cantidad, el sistema hace los respectivos cálculos para la transacción.




De click en el botón "Registrar Compra". Si el ingreso de datos es exitoso, se le mostrará un mensaje como el de la siguiente imagen.




Error en el ingreso

Los casos de errores se pueden presentar por datos inválidos, como el ingreso de un ID inexistente. Si se algún caso de error en los datos ingresado, el programa le mostrará el siguiente mensaje:




¿Qué tipo de datos debe ingresar?

A continuación se le presentara el orden preferente con el cual debe llenar los campos.

Primero ingrese el No. de Compra, este campo debe contener datos numéricos, de igual manera, el sistema no permite el ingreso desde teclado de otro tipo de dato.



Segundo llene el campo de proveedor, este le desplegara opciones de los proveedores existentes en el sistema. Al elegir cualquiera de estos se llenara el campo NIT.
NOTA: El campo NIT esta inhabilitado para el usuario.



Tercero, llene el campo fecha.
