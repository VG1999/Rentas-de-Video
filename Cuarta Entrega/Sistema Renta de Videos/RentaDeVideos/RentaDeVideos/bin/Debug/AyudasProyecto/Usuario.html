Ayuda: Control Usuarios


Acceso al Formulario Control Usuarios

Al dar clic en el submenú "Control Usuarios", en "Mantenimientos Usuarios". Usted podra gestionar cualquier información u operación con los Control Usuarios (ver la siguiente imagen).




Vista del Formulario

Al seleccionar "Control Usuarios" en el mantenimiento a continuación usted ingresará al formulario en donde encontrá todas las siguientes opciones posibles como ingresar, actualizar, eliminar y buscar.




Ingresar

Usted será redirigido al formulario de ingreso de datos para Control Usuarios, donde deberá llenar cada campo solicitado, para guardar los datos debe dar click en el boton "Guardar Datos en esquina inferior izquierda".




Formato para ingresar datos

A continuación se detalla en forma ilustrativa como debe ser el ingreso de datos para Control Usuarios (ver la imagen)




Al dar click en "Guardar Datos". Si el ingreso de datos es exitoso, se le mostrará un mensaje como el da la siguiente imagen




En el caso de errores se pueden presentar por datos inválidos, como el ingreso de un ID ya existente. Si se da algún error en los datos ingresados el programa le mostrará el siguiente error




Actualizar y Eliminar Datos

Usted será redirigido al formulario de Actualización y eliminación de datos del Control Usuarios, podra modificar y borrar cualquier dato seleccionando en la tabla que se muestra.




¿Como Actualizar?

Damos click en el campo que deseamos modificar o actualizar borramos el contenido y se sustituye por el dato deseado.




Si la actualización es exitosa, se le mostrará un mensaje como el da la siguiente imagen, de lo contrario el programa le notificara un error




¿Como Borrar?

Damos click derecho sobre el Control Usuarios que se desea borrar, le deplegará la opción de "Eliminar Dato". Recuerde que al ejecutar esta acción usted estará borrando todo la fila no solo un campo seleccionado.




Buscar

Al seleccionar "Buscar" usted será redirigido al formulario de busqueda de Control Usuarios donde usted podrá hacer busquedas por el nombre usuarios, id, entre otros como se muestra a continuación.



En la opcion columna le desplegará una lista de opciones por las cuales se pude realizar la busqueda, en el recuadro de texto usted deberá ingresar el dato que desea buscar como se muestra en la siguiente imagen.
